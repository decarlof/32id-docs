Detector
========

Open a terminal

First, check the status typing::

	[usertxm@txmtwo]$ pg status

Then, stop the IOC::

	[usertxm@txmtwo]$ pg stop

Restart the IOC::

	[usertxm@txmtwo]$ pg start
