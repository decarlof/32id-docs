Restarting ImageJ on txmone
===========================

Open a terminal::

    $ cd ~/ImageJ
    $ ./ImageJ.sh*

| In the ImageJ main window, go in the menu <plugins> <EPICS_areaDetector> <EPICS AD Viewer>

.. image:: img_guide/ImageJ.jpg
   :width: 600px
   :align: center
   :alt: project

| In the new window popping up, PVPrefix must be “32idcPG3:image1:”
| Press “Start”
